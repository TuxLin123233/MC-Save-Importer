## 项目简介

**Minecraft 存档管理器** 允许玩家轻松导入、导出和管理游戏存档，提供直观的用户界面和便捷的操作流程。

### 核心功能

- 📥 **导入存档**：将下载的 ZIP 格式地图解压到 `.minecraft/saves` 文件夹
- 📤 **导出备份**：将现有存档打包备份（开发中）
- 📋 **存档列表**：查看和管理所有存档（开发中）
- ❤️ **赞助支持**：支持开发者

## 技术栈

- **编程语言**：Python 3.10+
- **GUI 框架**：CustomTkinter
- **图像处理**：Pillow
- **打包工具**：PyInstaller

## 开始使用

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/main.py
```
