# 常见问题

## 导入存档相关问题

### Q: 找不到 .minecraft 文件夹

**A:** 请根据你的操作系统找到正确的路径：

- **Windows**: `C:\Users\<用户名>\AppData\Roaming\.minecraft`
- **Linux**: `~/.minecraft`
- **macOS**: `~/Library/Application Support/minecraft`

> 提示：Windows 用户可以在文件资源管理器地址栏输入 `%APPDATA%` 快速访问。

### Q: 提示"不是有效的.minecraft文件夹"

**A:** 有效的 `.minecraft` 文件夹应该包含 `launcher_profiles.json` 文件，以及以下至少一个：
- `saves` 文件夹（标准结构）
- `versions` 文件夹（版本迁移结构）

请确认选择的路径正确。

### Q: ZIP 文件解压失败

**A:** 可能的原因：
1. ZIP 文件已损坏，请重新下载
2. ZIP 文件内包含多个文件夹，请确保 ZIP 内只有一个存档文件夹
3. 磁盘空间不足

### Q: 导入时提示"存档已存在"

**A:** 这是正常提示。你可以选择：
- **覆盖**：用新存档替换旧存档
- **跳过**：保留旧存档，不导入该 ZIP

## 界面相关问题

### Q: 界面显示异常或字体错乱

**A:** 
1. 确保已安装 `HarmonyOS Sans` 字体（位于 `fonts/` 目录）
2. 检查 PIL 和 CustomTkinter 版本兼容性
3. 尝试删除 `__pycache__` 文件夹后重新运行

### Q: 窗口位置偏移或大小异常

**A:** 程序会自动居中窗口。如果仍有问题，请检查：
- 显示缩放设置（Windows：设置 > 系统 > 显示 > 缩放与布局）
- 多显示器环境下的主显示器设置

## 运行相关问题

### Q: 运行时报错缺少模块

**A:** 请安装所有依赖：

```bash
pip install -r requirements.txt
```

主要依赖：
- `customtkinter==5.2.2`
- `pillow==12.1.1`
- `playsound3`

### Q: 打包后的 EXE 无法运行

**A:** 
1. 确保打包命令中包含 `--add-data="../img;img"` 参数
2. 检查 `icon.ico` 文件是否存在
3. 尝试在命令行运行 EXE 查看错误信息

### Q: 音效无法播放

**A:** 
1. 确保 `sounds/` 目录存在且包含音频文件
2. 检查系统音量设置
3. 某些 Linux 发行版可能需要额外安装音频库

## 其他问题

### Q: 如何重置配置？

**A:** 删除项目根目录的 `data.json` 文件，程序会重新询问 `.minecraft` 路径。

### Q: 支持哪些 Minecraft 版本？

**A:** 本工具只操作存档文件，理论上支持所有 Java 版 Minecraft。存档格式兼容性取决于 Minecraft 版本本身。

### Q: 会修改或损坏我的存档吗？

**A:** 本工具只进行文件复制操作，不会修改存档内容。但建议：
- 导入前备份重要存档
- 不要在使用中的存档上进行操作

### Q: 如何反馈问题或建议？

**A:** 请在 GitHub 仓库提交 [Issue](../../issues)，描述：
1. 操作系统和版本
2. Minecraft 版本
3. 问题复现步骤
4. 错误信息（如有）
